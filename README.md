\# 🎵 SoundBox



Plataforma de gestión de música desarrollada en \*\*TypeScript\*\*.



\## Funcionalidades

\- Agregar canciones con título, artista, álbum, duración y género

\- Buscar canciones por título o artista

\- Filtrar canciones por género

\- Crear playlists y agregar/eliminar canciones

\- Ver duración total de una playlist

\- Ver estadísticas de la biblioteca



\## Tecnologías

\- TypeScript

\- Jest (pruebas unitarias)

\- GitHub Actions (CI/CD)



\## Instalación

```bash

npm install

npm test

```
